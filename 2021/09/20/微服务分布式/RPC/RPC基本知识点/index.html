分布式时主机间的通信方式

  TCP、UDP

  最基本的通信方式。二进制传输，其他所有通信方式，都是建立 在TCP或UDP之上。

  RPC，Remote Procedure Call

  PRC为远程方法调用。RPC本身是一个 概念 。重点是方法的调用，没有对象概念。

  Web Service

  基于Http + Xml的通信方式。

  REST，Representational State Transfer

  基于Http + JSON的通信方式。

  JMS，Java Message Service

  Java消息服务。将对象在物理意义上传输，相当于将对象，从客户端传输到服务器。

  RMI，Remote Method Invocation

  不能跨语言的Java远程方法调用。RMI也是一种RPC的实现方式。
